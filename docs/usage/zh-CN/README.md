## 推荐阅读顺序

1. 先看 `overview.md`，理解中央库工作流。
2. 再看 `skills-store.md` 和 `skills-management.md`，理解日常主流程。
3. 如果你管理多个工作流或项目本地技能，再看 `scenarios.md` 和 `project-workspaces.md`。
4. 当你需要配置环境、自定义 Agent 或备份时，再看 `settings.md` 和 `git-backup.md`。
